# Multisensa Physiotherapy - Express Application

This is the Express-based version of the Multisensa Physiotherapy website.

## Installation

1. Install dependencies:
```bash
npm install
```

## Running the Application

### Production Mode
```bash
npm start
```

### Development Mode (with auto-restart)
```bash
npm run dev
```

The server will start on http://localhost:3000
